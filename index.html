Recursos Poderosos

Tudo o que você precisa em um terminal moderno, com o poder da inteligência artificial.

🤖

IA Inline

Chat com IA diretamente no terminal. Tire dúvidas, obtenha sugestões de comandos e aprenda enquanto trabalha.

⚡

Performance

Construído com Electron e xterm.js para uma experiência rápida e responsiva, mesmo com grandes saídas.

🎨

Temas Personalizáveis

Escolha entre temas claros e escuros, ou crie o seu próprio para uma experiência única.

⌨️

Atalhos Inteligentes

Ctrl+I para IA, Ctrl+Shift+I para modo avançado. Navegue com eficiência usando atalhos intuitivos.

🔧

Multiplataforma

Disponível para Windows, macOS e Linux. A mesma experiência em qualquer sistema operacional.

🔒

Privacidade

Seus dados permanecem seguros. Configurações locais e controle total sobre suas informações.

Desenvolvido para Desenvolvedores

O Pnex é construído com tecnologias modernas e focado na experiência do desenvolvedor. Cada detalhe foi pensado para aumentar sua produtividade.

  • TypeScript para type safety
  • Integração com OpenAI
  • Terminal baseado em node-pty
  • Interface com Tailwind CSS
  • Código aberto e extensível
